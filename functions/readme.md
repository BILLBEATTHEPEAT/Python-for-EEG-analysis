### Introduction
Collection of usful functions for Analysis of EEG signals. They are gathered from several open-source project either in Github or self-pages.  
**Sallute to their own authors!**  
In Python 2.7

### Dependency
- Numpy
- Pandas
- PyEDFlib
- PyWavelets (pywt)
- Scipy
- PyEEG
- pyEntropy
